What If The Game
Actually Watches You?

We wanted to use AI to make games more immersive.

Right now, the player is invisible to the game. In most games, the player is always behind the screen, but the game knows almost nothing about them.

With visual input, we can understand where the player is looking, how they’re reacting, and how engaged they are.

We thought this is especially powerful in horror, because horror only works if the player feels immersed and tense.

So instead of just showing something scary, we wanted the game to react to the player themselves.

The idea is simple: what if the game actually watches you?

The Problem → Our Solution
